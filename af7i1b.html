문제

char, int, double 형 변수에 !, 15, 3.14의 값을 저장하고 포인터 변수에서 각 변수의 주소값을 
저장하고 기본 자료형 변수와 포인터 변수로 주소값 및 저장값 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
변수명   var주소값    저장값
---------------------------------------------
var1     0018FF44     !
var2     0018FF40     15
var3     0018FF38     3.140000
변수명   pnt주소값    var주소값    저장값
---------------------------------------------
pnt1     0018FF34     0018FF44     !
pnt2     0018FF30     0018FF40     15
pnt3     0018FF2C     0018FF38     3.140000
입력1)


변수명   var주소값    저장값
---------------------------------------------
var1     0018FF44     !
var2     0018FF40     15
var3     0018FF38     3.140000
변수명   pnt주소값    var주소값    저장값
---------------------------------------------
pnt1     0018FF34     0018FF44     !
pnt2     0018FF30     0018FF40     15
pnt3     0018FF2C     0018FF38     3.140000
입력2)


변수명   var주소값    저장값
---------------------------------------------
var1     0018FF44     !
var2     0018FF40     15
var3     0018FF38     3.140000
변수명   pnt주소값    var주소값    저장값
---------------------------------------------
pnt1     0018FF34     0018FF44     !
pnt2     0018FF30     0018FF40     15
pnt3     0018FF2C     0018FF38     3.140000

코드 빈칸 채우기

#include <stdio.h> 
	 
int main(void) 
{ 

	char   var1  = '!';	 
	int    var2  = 15;	 
	double var3  = 3.14;	 
 

	pnt1 = var1;				 
	pnt2 = var2;				 
	pnt3 = var3;				 
 
	printf("\n변수명   var주소값    저장값\n"); 
	printf("---------------------------------------------"); 
	printf("var1          %c\n", var1, var1 );				 
	printf("var2          %d\n", var2, var2 );				 
	printf("var3          %f\n", var3, var3 );				 
 
	printf("\n변수명   pnt주소값    var주소값    저장값\n"); 
	printf("---------------------------------------------\n"); 
	printf("pnt1               %c\n", pnt1, pnt1, pnt1 );		 
	printf("pnt2               %d\n", pnt2, pnt2, pnt2 );		 
	printf("pnt3               %f\n", pnt3, pnt3, pnt3 );		 
 
	return 0; 
}
