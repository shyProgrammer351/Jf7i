문제

국어 점수, 영어 점수, 수학 점수를 입력받아 평균을 계산하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
0점 ~ 100점 사이의 국어 점수를 입력하세요: 90
0점 ~ 100점 사이의 영어 점수를 입력하세요: 89
0점 ~ 100점 사이의 수학 점수를 입력하세요: 86
국어 90점, 영어 89점, 수학 86점의 평균은 88.33점입니다.


0점 ~ 100점 사이의 국어 점수를 입력하세요: 76
0점 ~ 100점 사이의 영어 점수를 입력하세요: 95
0점 ~ 100점 사이의 수학 점수를 입력하세요: 83
국어 76점, 영어 95점, 수학 83점의 평균은 84.67점입니다.


0점 ~ 100점 사이의 국어 점수를 입력하세요: 97
0점 ~ 100점 사이의 영어 점수를 입력하세요: 93
0점 ~ 100점 사이의 수학 점수를 입력하세요: 91
국어 97점, 영어 93점, 수학 91점의 평균은 93.67점입니다.

코드 빈칸 채우기

#include <stdio.h>  
 
void getScores( korean, english, mathematics, average )		 
{ 
	printf( "0점 ~ 100점 사이의 국어 점수를 입력하세요: " ); 
	( "%d", korean );		 
 
	printf( "0점 ~ 100점 사이의 영어 점수를 입력하세요: " ); 
	( "%d", english );				 
 
	printf( "0점 ~ 100점 사이의 수학 점수를 입력하세요: " ); 
	( "%d", mathematics );			 
 
	average = (double)( korean + english + mathematics ) / 3;			 
 
	return; 
} 
 
int main() 
{ 
	int korean = 0; 
	int english = 0; 
	int mathematics = 0; 
	double average = 0.0; 
	 
	getScores( korean, english, mathematics, average );				 
	 
	printf( "국어 %d점, 영어 %d점, 수학 %d점의 평균은 %.2f점입니다.", korean, english, mathematics, average ); 
	 
	return 0; 
}
