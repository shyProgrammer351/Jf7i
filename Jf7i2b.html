문제

영어 단어를 입력하면 영한 사전을 열고 그 뜻을 찾아서 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
--------- 파일 dictionary.txt의 내용 --------- 
apple의 뜻은 사과입니다.
banana의 뜻은 바나나입니다.
carrot의 뜻은 당근입니다.

영어단어를 입력하세요 : banana
banana의 뜻은 바나나입니다.


영어단어를 입력하세요 : apple
apple의 뜻은 사과입니다.


영어단어를 입력하세요 : carrot
carrot의 뜻은 당근입니다.

코드 빈칸 채우기

import java.util.*; 
import java.io.*; 
 
public class DictionaryMain  
{ 
	public static void main( String[] args )  
	{ 
		Scanner scan = new Scanner( System.in );	 
		String word = ""; 
		String meaning = ""; 
 
		System.out.println( "영어단어를 입력하세요 : " ); 
		word = scan.nextLine();  
		try 
		{ 
			 dictionary = new ( new ( new ( "C:\\Users\\user\\Downloads\\dictionary.txt" ) ) ); 	 
			while ( ( meaning = dictionary.() ) != null )  
			{ 
				if ( meaning.( word ) == 0 )  
				{ 
					System.out.println( meaning ); 
					break; 
				} 
			} 
		 
			dictionary.();	  
		} 
		catch (  e )  
		{ 
			System.out.println( "[오류] 파일을 열 수 없습니다!" ); 
			e.printStackTrace(); 
		} 
 
		return; 
	} 
}
