문제

국어 점수, 영어 점수, 수학 점수를 입력받아 평균을 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
0점 ~ 100점 사이의 국어 점수를 입력하세요: 90
0점 ~ 100점 사이의 영어 점수를 입력하세요: 89
0점 ~ 100점 사이의 수학 점수를 입력하세요: 86
국어 90점, 영어 89점, 수학 86점의 평균은 88.33점입니다.


0점 ~ 100점 사이의 국어 점수를 입력하세요: 76
0점 ~ 100점 사이의 영어 점수를 입력하세요: 95
0점 ~ 100점 사이의 수학 점수를 입력하세요: 83
국어 76점, 영어 95점, 수학 83점의 평균은 84.67점입니다.


0점 ~ 100점 사이의 국어 점수를 입력하세요: 97
0점 ~ 100점 사이의 영어 점수를 입력하세요: 93
0점 ~ 100점 사이의 수학 점수를 입력하세요: 91
국어 97점, 영어 93점, 수학 91점의 평균은 93.67점입니다.

알고리즘

 프로그램 시작 
    과목별 점수 및 평균을 구하기 
    과목별 점수 및 평균을 출력 
 프로그램 종료 
 
 과목별 점수 및 평균을 구하는 함수 시작 
    국어 점수를 입력 
    영어 점수를 입력 
    수학 점수를 입력 
    평균 점수를 계산 
 과목별 점수 및 평균을 구하는 함수 종료

프로그램 코드 

	// 파일명 : ./Chapter07/Score.c 

	#include <stdio.h>  

	 

	// 과목별 점수 및 평균을 구하는 함수 시작 

1	void getScores( int *korean, int *english, int *mathematics, double *average )		 

	{ 

		// 국어 점수를 입력 

		printf( "0점 ~ 100점 사이의 국어 점수를 입력하세요: " ); 

		// 90점을 입력한다고 가정 

		scanf( "%d", korean );		 

	 

		// 영어 점수를 입력 

		printf( "0점 ~ 100점 사이의 영어 점수를 입력하세요: " ); 

		// 89점을 입력한다고 가정 

		scanf( "%d", english );				 

	 

		// 수학 점수를 입력 

		printf( "0점 ~ 100점 사이의 수학 점수를 입력하세요: " ); 

		// 86점을 입력한다고 가정 

		scanf( "%d", mathematics );			 

	 

		// 평균 점수를 계산 

		*average = (double)( *korean + *english + *mathematics ) / 3;			 

	 

		// 과목별 점수 및 평균을 구하는 함수 종료 

		return; 

2	} 

	 

	// 프로그램 시작 

3	int main() 

	{ 

		// 점수 초기화 

		int korean = 0; 

		int english = 0; 

		int mathematics = 0; 

		double average = 0.0; 

		 

		// 과목별 점수 및 평균을 구하기 

4		getScores( &korean, &english, &mathematics, &average );				 

		 

		// 과목별 점수 및 평균을 출력 

5		printf( "국어 %d점, 영어 %d점, 수학 %d점의 평균은 %.2f점입니다.", korean, english, mathematics, average ); 

		 

		// 프로그램 종료 

6		return 0; 

	}
