문제

국어 점수, 영어 점수, 수학 점수를 입력받아 평균을 계산하는 문제입니다 실행순서를 클릭하세요 
0점 ~ 100점 사이의 국어 점수를 입력하세요: 90
0점 ~ 100점 사이의 영어 점수를 입력하세요: 89
0점 ~ 100점 사이의 수학 점수를 입력하세요: 86
국어 90점, 영어 89점, 수학 86점의 평균은 88.33점입니다.

프로그램 코드

	#include <stdio.h> 


1	void getScores( int *korean, int *english, int *mathematics, double *average )		

	{

		printf( "0점 ~ 100점 사이의 국어 점수를 입력하세요: " );

		scanf( "%d", korean );		


		printf( "0점 ~ 100점 사이의 영어 점수를 입력하세요: " );

		scanf( "%d", english );				


		printf( "0점 ~ 100점 사이의 수학 점수를 입력하세요: " );

		scanf( "%d", mathematics );			


		*average = (double)( *korean + *english + *mathematics ) / 3;			


		return;

2	}


3	int main()

	{

		int korean = 0;

		int english = 0;

		int mathematics = 0;

		double average = 0.0;

		

4		getScores( &korean, &english, &mathematics, &average );				

		

5		printf( "국어 %d점, 영어 %d점, 수학 %d점의 평균은 %.2f점입니다.", korean, english, mathematics, average );

		

6		return 0;

	}
